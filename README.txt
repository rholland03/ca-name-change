California Name Change After Marriage Website
-------------------------------------------------
This static site provides resources for changing your name after marriage in California.

Files:
- index.html: Main HTML page with step-by-step instructions and comment section
- styles.css: Site styling with visual enhancements
- script.js: Script to load and toggle county data
- counties.json: County Clerk info (all 58 counties, placeholder data)
- README.txt: This file

Instructions to host:
1. Unzip the folder.
2. Upload all files to GitHub or another web host.
3. Enable GitHub Pages (Settings → Pages → Branch: main → Folder: root).
4. Visit the provided GitHub Pages URL to see your live site.
